MONTREAL HELP FILE

Thank you for purchasing Montreal, I have tried my best to make it easy for you to use. If you still need help with the template feel free to get in touch.

SEND EMAIL

Okay first things first for you to be able to make the contact form and project planner work you need to edit the php files with your email. Open up sendemail.php and look for this section at the beginning of the file.

// change this email address to your own email id. define("CONTACT_EMAIL", 'youremail@domain.com');

Now open up projectplanner.php and look for this in the begining of the file

$fromemail="Project Planner "; // change here if you want $toemail="youremail@domain.com"; // change here EMAIL ADDRESS

Also open up js folder and main.js file. Change the twitter username from envato to your own.

each row should start with class="row" to keep all columns align properly. But if you would like to avoid using class="row" you can add class="alpha" to a column to make it the first column of a row. class="four columns alpha" will remove left padding from four columns div to align it to left and make it the first column block of the row.

you can also align columns to center. Using class="centered" to columns will align div to center. e.g class="eight columns centered"

HOMEPAGE SLIDESHOW

Montreal uses supersized for out homepage slideshow. To modify the slideshow images, open up js folder and open slideshow.js scroll down to where it says

{image : '../preview/bg.jpg', thumb : '../preview/bg.jpg'}, and change the link to your images for slideshow.

PADDING AND MARGIN

I have tried my best to make modifying or creating layouts as easy as possible without having to modify or add css or doing any hardline coding. For this reason I have added various CSS classes for paddings and margins. Adding any of the following classes will give each element its respective padding or margin. These class shortcodes makes it very easy to create a beautiful layout

Padding shortcodes

  • class=smallpadding > 10px top bottom
  • class=smallsidepadding > 10px left right
  • class=smalltoppadding > 10px top
  • class=smallbottompadding > 10px bottom
  • class=midpadding > 20px top bottom
  • class=midtoppadding > 20px top
  • class=midbottompadding > 20px bottom
  • class=bigpadding > 50px top bottom
  • class=bigsidepadding > 50px left right
  • class=bigtoppadding > 50px top
  • class=bigbottompadding > 50px bottom
  • class=largepadding > 100px top bottom
  • class=largetoppadding > 100px top
  • class=largebottompadding > 100px bottom
  • class=leftpadding > 20px left
  • class=rightpadding > 20px right

Margin shortcodes

  • class=smallmargin > 10px top bottom
  • class=smalltopmargin > 10px top
  • class=smallbottommargin > 10px bottom
  • class=midmargin > 20px top bottom
  • class=midtopmargin > 20px top
  • class=midbottommargin > 20px bottom
  • class=bigmargin > 50px top bottom
  • class=bigtopmargin > 50px top
  • class=bigbottommargin > 50px bottom
  • class=largemargin > 100px top bottom
  • class=largetopmargin > 100px top
  • class=largebottommargin > 100px bottom

TEXT STYLES

HOW TO USE OTHER PORTFOLIOS ON HOMEPAGE

Because all pages are commented at most it would be a copy paste to add or subtract items. However if you want to change the default portfolio on homepage. You can simply copy portfolio items , but since it can be a bit confusing the below code should help. We will skip past the basic portfolio since its already on display.

Grid Portfolio

<section class="row">
<!-- PROJECT ONE -->
<div class="six columns griditem">
<img src="IMAGE URL" alt="griditem"/>
<a href="/PAGE URL" title="Click To View Project">
<div class="gridinfo"> <h3 class="whitetext extrabold bigtoppadding center">PROJECT TITLE</h3>
<h5 class="whitetext center">WEB DESIGN / UI/ UX</h5>
<span class="smallfont space">VIEW PROJECT</span>
</div>
<div class="gridblack">
<h2 class="whitetext">WEBDESIGN <br/> USER INTERFACE <br/> USER EXPERIENCE</h2>
</div>
</a>
</div>
</section>
'gridblack shows after 1.6s hover over project. Also works good for SEO. Also note we start each columns row with class=row and as said before each row can have a maximum of 12 columns and in this example two six columns griditems'

Grid Block Portfolio

<div class="container nopadding gridder">
<!-- PROJECT ONE -->
<div class="gridblock">
<img src="IMAGE URL" alt="griditem"/>
<a href="/PAGE URL" title="Click To View Project">
<div class="gridinfo">
<h3 class="whitetext extrabold smalltoppadding center">PROJECT TITLE</h3>
<h5 class="whitetext center">WEB DESIGN / UI/ UX</h5>
<span class="smallfont space whitetext">VIEW PROJECT</span>
</div>
</a>
</div>
In the above example we add class nopadding and gridder to container. This makes the portfolio grid spread wide without margins and padding. We dont use row as each gridblock floats to left.

<div class="drag">
